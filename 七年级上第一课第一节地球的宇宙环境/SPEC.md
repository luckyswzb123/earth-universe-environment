# 《地球的宇宙环境》交互式课件 开发文档

## 1. 项目概述

### 1.1 项目背景
- **项目名称**: 地球的宇宙环境 - 交互式课件
- **目标用户**: 七年级上学期地理教师与学生
- **项目类型**: 教育类Web交互式课件（MVP版本）
- **核心目标**: 通过可视化交互帮助学生理解太阳系结构和地球宜居性

### 1.2 技术栈
- **前端框架**: 原生HTML5 + CSS3 + JavaScript（ES6+）
- **动画技术**: CSS动画 +少量JavaScript控制
- **图形渲染**: SVG（用于太阳系模型）
- **无外部3D依赖**: 纯CSS实现2.5D视觉效果

### 2.1 模块一：太阳系模型

#### 2.1.1 页面结构
```
┌─────────────────────────────────────────┐
│              太阳系模型区域               │
│     ☀️ (中心)                           │
│   ○轨道1  ○轨道2  ○轨道3 ...            │
│  水星 金星 地球 火星 木星 土星 天王 海王  │
│                                         │
│  ┌─────────────────┐  ┌──────────────┐  │
│  │  控制面板        │  │  信息卡片    │  │

│  │  [比例说明按钮] │  │              │  │
│  └─────────────────┘  └──────────────┘  │
└─────────────────────────────────────────┘
```

#### 2.1.2 功能规格

| 功能 | 描述 | 交互行为 |
|------|------|----------|
| 太阳系示意图 | SVG绘制的3D8大行星+3D太阳 | 动态展示 |
| 行星悬停效果 | 鼠标悬停时行星放大1.2倍 | :hover + transform |
| 行星名称显示 | 悬停时显示行星名称标签 | fadeIn动画 |
| 信息卡片弹窗 | 点击行星弹出详细信息 | 模态框/气泡 |
| 轨道显示开关 | 滑块控制轨道线显示/隐藏 | CSS类切换 |
| 比例说明 | 点击显示实际比例文字说明 | toggle显示 |

#### 2.1.3 行星数据（信息卡片内容）
| 行星 | 直径(km) | 与太阳距离(百万km) | 表面温度(℃) |

| 金星 | 12,104 | 108.2 | 465 |
| 地球 | 12,742 | 149.6 | -88~58 |
| 火星 | 6,779 | 227.9 | -87~-5 |
| 木星 | 139,820 | 778.6 | -110 |

#### 2.1.4 实现要点

- 行星颜色按真实颜色渲染（水星灰色、金星黄色、地球蓝色等）

### 2.2 模块二：宜居带演示器

#### 2.2.1 页面结构
```
┌─────────────────────────────────────────┐
│            宜居带演示器                   │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │   🔴(太热) 🟢(宜居) 🔵(太冷)    │  │
│  │     ◐◐◐◐◐◐◐◐◐◐◐◐◐◐◐◐◐◐      │  │
│  │                                   │  │
│  │              ☀️                    │  │
│  │                                   │  │
│  │              🌍 [可拖拽]           │  │
│  └───────────────────────────────────┘  │
│                                         │
│  ┌─────────────────┐  ┌──────────────┐  │
│  │  当前区域: 宜居  │  │  温度: 15℃   │  │
│  │  水状态: 液态    │  │              │  │
│  │  结论: 适合生命  │  │              │  │
│  └─────────────────┘  └──────────────┘  │
└─────────────────────────────────────────┘
```

#### 2.2.2 功能规格

| 功能 | 描述 | 交互行为 |
|------|------|----------|
| 太阳居中 | 太阳图标固定在中间 | 静态 |
| 三色区域 | 红/绿/蓝渐变区域表示温度 | CSS gradient |
| 地球拖拽 | 可拖动地球图标改变位置 | drag事件 |
| 区域检测 | 实时检测地球所在区域 | 距离计算 |
| 温度计算 | 简化公式计算温度 | 线性映射 |
| 水状态显示 | 根据温度显示水状态 | 条件判断 |
| 结论显示 | "适合生命/不适合"文字 | 状态切换 |

#### 2.2.3 温度计算公式（简化版）
```
距离太阳越远，温度越低
温度 = 100 - (距离 / 最大距离) * 300
```

| 区域 | 温度范围 | 水状态 | 结论 |
|------|----------|--------|------|
| 红色(太热) | >60℃ | 气态 | 不适合 |
| 绿色(宜居) | 0~60℃ | 液态 | 适合生命 |
| 蓝色(太冷) | <0℃ | 冰 | 不适合 |

#### 2.2.4 实现要点
- 使用CSS `linear-gradient`创建三色背景
- 地球拖拽使用原生JavaScript `mousedown/mousemove/mouseup`
- 温度实时更新使用 `requestAnimationFrame`优化性能

| 2013 | 嫦娥三号 | 月球车图 | 嫦娥三号探测器成功着陆月球，玉兔号月球车开始月面巡视探测 | 玉兔号在月球上工作了972天 |

#### 2.3.4 实现要点
- 使用CSS `display: flex`实现水平布局
- 节点使用 `::before/::after`伪元素绘制
- 内容区使用 `display: none/block`切换

## 3. UI/UX 设计规范

### 3.1 配色方案
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色调 | 深空蓝 | #1a1a2e |
| 强调色 | 科技蓝 | #0f3460 |
| 文字色 | 白色 | #ffffff |
| 辅助色 | 金色 | #e6b800 |
| 背景色 | 星空黑 | #0a0a15 |
| 成功色 | 绿色 | #00c853 |
| 警告色 | 红色 | #ff1744 |
| 寒冷色 | 蓝色 | #2979ff |

### 3.2 字体规范
- **标题**: "Noto Sans SC", sans-serif, 24px~32px
- **正文**: "Noto Sans SC", sans-serif, 14px~16px
- **数字**: "DIN Alternate", sans-serif（用于数据展示）

### 3.3 间距系统
- 基础单位: 8px
- 小间距: 8px
- 中间距: 16px
- 大间距: 24px
- 区块间距: 32px

### 3.4 响应式断点
| 设备类型 | 宽度 | 布局调整 |
|----------|------|----------|
| 桌面 | >1024px | 双栏布局 |
| 平板 | 768~1024px | 单栏+收缩 |
| 手机 | <768px | 垂直堆叠 |
